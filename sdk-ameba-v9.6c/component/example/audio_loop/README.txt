This audio loop example is used to show how to use the audio system with A-mic or D-mic.

	1. Use -DEXAMPLE=audio_loop to gearte Make file 
	
	2. Modify the flag CONFIG_MIC_TYPE USE_DMIC to enable D-mic, while CONFIG_MIC_TYPE USE_AMIC to enable A-mic
	
	3. Build and flash the binary to test
