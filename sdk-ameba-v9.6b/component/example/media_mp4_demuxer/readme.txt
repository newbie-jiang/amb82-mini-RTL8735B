
This example is for mp4 demuxer.
It show how to use the example to demuxer the mp4 video.
Please reference the follow steps
1.Put your mp4 video to sdcard.The name is AMEBA_PRO.mp4.
2.Run the example and you can see the raw video and audio data.

Noted:
The video only support AMEBAPRO2 device.
